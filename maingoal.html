Main Goal:

The main goal of this project is to make sure that all mobile users, including prepaid customers, get the most out of their data plans. It aims to prevent any unused data from going to waste. For prepaid users, the project ensures that their leftover data doesn't just disappear but can be saved for future use. Moreover, this unused data can be shared or sold at lower rates to others, reducing costs for everyone. In simpler terms, the main goal is to ensure that no data is wasted, and more people can benefit from it.

  • Data Conservation: Preserve unused data for prepaid users, ensuring they don't lose what they've paid for.
  • Fairness: Create a fair system where both prepaid and postpaid users can roll over unused data, promoting equity.
  • Data Sharing: Enable users to share or sell their excess data to others, reducing costs for everyone.
  • User Experience: Prioritize a user-friendly interface for managing and accessing rolled-over data.
  • Scalability: Create a system that can handle a growing number of users and increasing amounts of rolled-over data as the project expands.
